# Markdown to PDF
Наверное один из самых старых проектов который пришлось заного пересоздавать, так как старая версия была не идеальной.
Я уверен что данный инструмент поможет многим школьникам и студентам, которые выполняют дз через файлы вроде PDF или DOCX.
## А что это за инструмент то?
> Если кратко....
> Если говорить по простому, то данный сайт переформулирует текст написаный через markdown в документ типа PDF, в скором времени планируется добавить так же DOCX, но это когда нибудь по позже.

В частности хочется сказать что данный сайт не был вручную написан лично мною, а был переделан, и улучшен через такую нейросеть как Qwen3-Max (шуток про месенджер не будет).
