# Markdown to PDF

Инструмент для конвертации текста в формате Markdown в PDF документ.

## Что такое Markdown

Markdown — это простой способ форматирования текста, который используется во многих популярных сервисах: искусственный интеллект (ChatGPT, Claude и другие), Discord, Telegram, GitHub, Reddit и других платформах.

Вместо сложных кнопок и меню вы используете специальные символы для оформления: звёздочки для жирного текста, решётки для заголовков, дефисы для списков. Это позволяет быстро писать и сразу видеть результат.

## Для кого этот инструмент

Этот конвертер особенно полезен для студентов, которые выполняют задания онлайн. Если вы пишете ответы на вопросы, рефераты или курсовые работы не в тетрадях, а в электронном виде, то этот инструмент поможет вам получить красиво оформленный документ, который можно распечатать или отправить преподавателю.

Также инструмент подойдёт всем, кто работает с текстом: писателям, блогерам, специалистам технической поддержки, которые готовят инструкции, и многим другим.

## Как использовать

Откройте файл index.html в любом браузере. Слева находится редактор, куда нужно вставить ваш текст в формате Markdown. Справа вы сразу видите, как будет выглядеть итоговый документ.

Напишите текст, используя простое форматирование: решётка перед текстом создаст заголовок, звёздочки вокруг слова сделают его жирным, дефис в начале строки создаст элемент списка. Когда закончите, нажмите кнопку Скачать PDF внизу страницы. Файл сохранится на вашем компьютере.

## Возможности

Редактор поддерживает все основные элементы Markdown: заголовки разных уровней, жирный и курсивный текст, списки, ссылки, изображения, код, цитаты и таблицы. Вы можете выбрать шрифт и размер текста для предпросмотра.

Для работы нужен только современный браузер. Интернет требуется только для загрузки необходимых библиотек при первом открытии.
