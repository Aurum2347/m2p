# Markdown to PDF Конвертер

Этот инструмент позволяет легко преобразовать текст в формате Markdown в PDF документ.

## Что такое Markdown

Markdown — это простой формат текста, который используется во многих популярных сервисах: искусственный интеллект (ChatGPT и другие), Discord, Telegram, GitHub и многих других платформах. Он позволяет оформлять текст с помощью простых символов: звёздочки для жирного текста, решётки для заголовков, дефисы для списков и так далее.

## Для кого этот инструмент

Этот конвертер особенно полезен для студентов, которые выполняют задания онлайн и хотят получить красиво оформленный документ для сдачи преподавателю. Вместо того чтобы вручную форматировать текст в Word или другом редакторе, вы можете написать его в простом формате Markdown и сразу получить готовый PDF файл.

## Как использовать

1. Откройте файл index.html в любом браузере
2. В левой части введите ваш текст в формате Markdown
3. Справа вы сразу увидите, как будет выглядеть ваш документ
4. Настройте шрифт и размер текста по желанию
5. Нажмите кнопку Скачать PDF чтобы получить готовый файл

## Возможности

- Мгновенный предпросмотр результата
- Поддержка всех основных элементов Markdown: заголовки, списки, таблицы, код, изображения, ссылки
- Выбор шрифтов для текста и кода
- Настройка размера шрифта
- Тёмная и светлая тема оформления
- Экспорт в PDF одним кликом
- Предпросмотр PDF перед скачиванием

## Требования

Для работы конвертера нужен только современный браузер (Chrome, Firefox, Safari, Edge). Никаких дополнительных программ устанавливать не требуется.

## Примечание

Все данные обрабатываются только в вашем браузере и никуда не отправляются. Вы можете работать офлайн после первой загрузки страницы.
