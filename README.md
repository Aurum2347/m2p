# Markdown → PDF

Простой инструмент для преобразования текста в формате Markdown в готовый PDF-документ.

## Что такое Markdown?

Markdown — это лёгкий способ форматирования текста с помощью простых символов. Вы наверняка уже встречали его, даже если не знали названия:

- Нейросети (Qwen, DeepSeek и другие) часто отвечают в этом формате
- Discord отображает сообщения с Markdown-разметкой
- GitHub использует Markdown для README-файлов и документации
- Telegram поддерживает базовое форматирование в стиле Markdown

Вместо сложных кнопок вы пишете `**жирный**`, `*курсив*`, `# Заголовок` — и текст сразу выглядит как надо.

## Как пользоваться

Интерфейс интуитивный и разделён на три части:

📝 **Слева — редактор**  
Пишите или вставляйте ваш Markdown-текст. Поддерживаются заголовки, списки, ссылки, изображения, код, таблицы и даже блок-схемы.

👁️ **Справа — предпросмотр**  
В реальном времени видите, как будет выглядеть ваш документ. Меняете текст слева — сразу видите результат справа.

⬇️ **Кнопка «PDF»**  
Нажмите, чтобы скачать готовый документ. Укажите имя файла и выберите размер шрифта — всё просто.
