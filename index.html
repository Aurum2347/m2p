Markdown to PDF

Профессиональный инструмент для преобразования Markdown в PDF с поддержкой:

  • Форматирования Markdown
  • Блок-схем и графиков (DOT/Graphviz)
  • Формул LaTeX через KaTeX
  • Гибких настроек экспорта
Используйте ```dot ... ``` для создания диаграмм Graphviz.
