Markdown to PDF

Профессиональный инструмент для преобразования Markdown в PDF с расширенными возможностями:

  • Полная поддержка Markdown форматирования
  • Рендеринг математических формул через KaTeX
  • Поддержка DOT-диаграмм (Graphviz)
  • Светлая и тёмная темы
  • Экспорт в PDF с сохранением форматирования
  • Встроенный поиск по документу
  • Автоматическое форматирование таблиц

Используйте ```dot ... ``` для создания диаграмм Graphviz.

Нажмите Ctrl+F для поиска по документу.
